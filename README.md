# Nursery-Management-System
A web-based Nursery Management System developed using Java, JSP, Servlets, JDBC, and MySQL.  The project includes user authentication, product listing, cart management, order placement,  and session handling. Developed as part of an academic project.

# Nursery Management System

Technology Used:
- Java 17
- JSP & Servlets
- JDBC
- MySQL
- HTML, CSS, JavaScript

Features:
- User Login & Logout
- Product Listing
- Cart using LocalStorage
- Checkout & Order Placement
- View Orders
- Session Management

Developed as part of MCA project.
